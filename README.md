## ğŸš€ Quick Start

### Prerequisites
- Node.js (v14 or higher)
- Python (v3.8 or higher)
- Git

### Installation
```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/synergai.git
cd synergai

# Install dependencies
# Frontend
cd frontend
npm install

# Backend
cd backend
npm install

# AI Service
cd ai-service
pip install -r requirements.txt
```

### Run Locally
```bash
# Start frontend (Terminal 1)
cd frontend
npm start
# Visit: http://localhost:3000

# Start backend (Terminal 2)
cd backend
npm start
# Runs on: http://localhost:5000

# Start AI service (Terminal 3)
cd ai-service
python main.py
# Runs on: http://localhost:8000
```
